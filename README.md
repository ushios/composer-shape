Installation
=============

Using composer
---------------

```
{
    ...
    "require": {
        "ushios/shape": "0.*"
    }
}
```

[@see Packagist](https://packagist.org/packages/ushios/shape)

===========

Test
-----

```
$ vendor/bin/phpunit
```

Format
-------

```
$ vendor/bin/php-cs-fixer fix 
```

Make API Document using phpDocumenter
--------------------------------------

$ vendor/bin/phpdoc
